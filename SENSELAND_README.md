This branch of NASA's EVHR processing code was created in order to better assist the specific purposes of the researchers at the SensLand Lab (https://www.senslandlab.com/) at Kent State University.

Here, we merge the EVHR (Enhanced Very-high Resolution) extension with it's underlying dependenc (core). This allows users of this repo to forgo some dependency management.

Further documention and references can be found in the file titled README.md in this same folder. All credit goes to the original developers of the project.
